# 📊 Ads Performance Dashboard

Google Ads・Meta Ads・TikTok Adsの広告パフォーマンスをCSVから即座に可視化するダッシュボードです。
媒体をまたいだKPI比較から、キャンペーン単位の詳細分析まで、広告運用に必要な数値を一画面で把握できます。

## 📊 主な機能

### CSVアップロード・カラムマッピング
- 任意のCSVをアップロードしてカラムを手動でマッピング
- 広告費・CV数・日付・キャンペーンタイプ・売上・表示回数・クリック数に対応
- ファイル名からGoogle Ads / Meta Ads / TikTok Adsを自動判定
- ROAS・CPA・CVRはマッピング後に自動計算
- 通貨記号（$・¥・なし）を選択可能。¥選択時はK省略せずカンマ表記

### KPIカード
- 総広告費・総CV数・平均CPA・平均ROASをリアルタイム表示
- フィルター連動で選択媒体のKPIに即時更新

### マルチ・シングルプラットフォーム対応
- **複数媒体データ**：媒体別ROAS比較・広告費シェア・月別トレンドを表示
- **単一媒体データ**：日別パフォーマンス推移・キャンペーン別詳細テーブルを表示
- データに応じて表示するグラフを自動切替（不要なグラフは非表示）

### 日別パフォーマンス推移グラフ
- 広告費・CPA・CVR・ROASをタブ切替で1グラフに表示
- ホバーで数値ツールチップを表示
- 期間フィルター付き（CSVの最古〜最新日付で初期値を自動設定）

### キャンペーン別パフォーマンステーブル
- 広告費・CV数・CPA・表示回数・クリック数・CVR・収益・ROASを一覧表示
- 全列ヘッダークリックで昇順/降順ソート
- 横スクロール対応で列が多くても見やすい
- 期間フィルターで任意の日付範囲に絞り込み可能

### その他
- プラットフォームフィルターボタンで媒体別に全グラフ・KPIを一括切替
- ダークモード / ライトモード切替
- スマホ・タブレット対応のレスポンシブレイアウト

## 💡 技術的な工夫

- **グラフの動的制御**：データが1種類しかない場合など、意味のないグラフは自動で非表示にすることでUIをクリーンに保つ
- **CSVフォーマット吸収**：`DD-MM-YYYY` / `MM/DD/YYYY` / `YYYYMMDD` など複数の日付フォーマットを正規化して処理
- **重複イベントリスナー防止**：再描画のたびにクローン差し替えパターン（`cloneNode`）でリスナーの多重登録を防止
- **プラットフォーム自動判定**：ファイル名の正規表現マッチングと列値の正規化を組み合わせ、表記ゆれ（`fb` / `Facebook` / `instagram` など）に対応
- **キャンペーン名の名寄せ**：英数字のみ抽出・先頭6文字比較でタイポや表記揺れがある同名キャンペーンを同一視
